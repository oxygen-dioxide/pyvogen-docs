# 安装与配置
## Python的安装
python可以在[python官网](https://www.python.org/)下载。

如果没有管理员权限，请在[微软商店](https://www.microsoft.com/en-us/p/python-39/9p7qfqmjrfp7)下载。

## PyVogen的安装
在命令行中运行：
```
pip install vogen 
```

## 安装音源
命令行安装：
```
pyvogen pm install <音源包路径>
```
安装完成后，用以下命令查看已安装音源：
```
pyvogen pm list
```

也可以在python内安装
```py
from vogen import pm
pm.install("音源路径")
```
